## Overview
I designed and developed a full-stack multi-tenant SaaS/ERP platform for a multi-branch dance school organization. The system replaces spreadsheet-based workflows by centralizing student records, scheduling, financial tracking, and reporting into a single web application.

The platform is used operationally by staff across different locations.

Live Demo:
https://studio-management-sys-production.up.railway.app


---

## Core Features
- Student registration and management
- Course enrollment (master and optional courses)
- Class scheduling and camp planning
- Attendance, breaks, and withdrawals tracking
- Financial transactions and payment tracking
- Monthly revenue and unpaid student monitoring
- Audit logs recording user actions

---

## SaaS Administration
- Super Admin control center
- Role-based access and permissions
- Branch management
- System configuration
- Subscription control

The platform integrates PayPal subscription billing and verifies transactions via webhooks.

---

## Technology Stack

Frontend:
- React
- Tailwind CSS
- JavaScript (ES6)

Backend:
- Node.js
- Express.js REST API

Database:
- MySQL relational database

Infrastructure:
- Cloud deployment
- Authentication and authorization
- Multi-tenant data isolation
